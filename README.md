# JavaScript To-Do Project (LocalStorage)

This is a simple, responsive **To-Do application** built with **HTML, CSS and vanilla JavaScript**.
It uses **localStorage** to save tasks in the user's browser — no backend or server required.

## Features
- Add tasks
- Edit tasks (double-click or press ✎)
- Mark complete / undo
- Delete tasks
- Filter (All / Active / Completed)
- Search tasks
- Clear completed tasks
- Drag to reorder tasks
- Saved in browser (localStorage)
- Mobile responsive

## How to run
1. Unzip the project.
2. Open `index.html` in your browser (double-click file).
3. Or push to GitHub and host via GitHub Pages (static site).

## Files
- `index.html` — main page
- `style.css` — styling
- `script.js` — app logic
- `README.md` — this file

## Notes for submission
- No server required.
- Works offline in the browser (data saved in localStorage).
- To reset data: open browser DevTools → Application → Local Storage → delete the key `todo_tasks_v1`.
